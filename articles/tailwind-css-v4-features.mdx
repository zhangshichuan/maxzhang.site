# Tailwind CSS v4 新特性概览

这是关于 **Tailwind CSS v4** 的文章内容。

更快的构建速度，更强大的 CSS 变量支持，Tailwind v4 来了。

## 更快的构建速度
Tailwind CSS v4 致力于提供更极致的构建性能，通过优化内部算法和与构建工具的集成，显著缩短开发和生产环境的编译时间。

<h2>强大的 CSS 变量支持</h2>
v4 版本进一步深化了对 CSS 变量的利用，使得主题定制和动态样式更加灵活和强大。开发者可以更方便地通过 CSS 变量来控制设计系统。

<h2>新的插件架构</h2>
全新的插件架构使得扩展 Tailwind CSS 变得前所未有的简单，社区可以更轻松地创建和分享自定义功能。
